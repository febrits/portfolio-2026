# SnippetVault

A code snippet manager with syntax highlighting and tagging.

## Features

- Create, organize, and search code snippets
- Syntax highlighting for 50+ languages
- Tagging system
- Favorite snippets
- Copy to clipboard with one click
- Dark/Light mode
- Responsive design

## Tech Stack

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- (Future: PostgreSQL, Prisma, NextAuth)

## Getting Started

```bash
npm install
npm run dev
```
